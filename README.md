# WebsocketServer
    High availability and high performance websocket communication service

# Quick Start
    1. Create dependent directory
        mkdir -p git.nzajiw.com/zb

    2. Download websocket server
        cd git.nzajiw.com/zb
        git clone http://192.168.88.205:1080/zb/websocketserver.git 

    3. Compile
        cd websocketserver
        sh -c build.sh

    4. Run
        sh -c start.sh
